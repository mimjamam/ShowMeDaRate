# ShowMeDaRate: Currency Conversion Chatbot

A simple Flask-based backend that integrates with Dialogflow to provide real-time currency conversion, with a web frontend using Dialogflow Messenger.

**Features:**
- Connects to Dialogflow ES as a webhook for currency conversion.
- Uses [exchangerate-api.com](https://www.exchangerate-api.com/) for live rates.
- Includes a modern web UI powered by Dialogflow Messenger.
- Ready for deployment on Render, Railway, Heroku, or any modern Python web host.
